# peaceofrealestate.com

## Overview

Peace of Real Estate (PRE) is a bilateral matching platform built to connect
real estate consumers with agents based on working style, communication
expectations, transparency, and overall fit.

Core idea:

- Consumers get matched with agents who fit how they want to buy or sell.
- Agents get introduced to consumers who align with how they work.
- Matching is based on responses from both sides, not a one-sided lead form.

## Landing Page

The most expensive decision of your life starts with one choice: the right

PRE is the first platform built to make sure that choice is actually right,
matched on working style, communication expectations, and fit. Built for the
consumer and the transparent, authentic agent simultaneously.

## User Flows

The platform has two distinct but connected journeys:

1. Consumer journey
2. Agent journey

Each side completes its own intake flow, and the platform uses both sets of
responses to generate a bilateral fit score.

### Consumer Flow

- Focus: help consumer find best-fit agent
- Cost: free entry, optional paid upgrade
- Outcome: ranked agent list, then direct introduction after selection

1. **Category weighting**: Consumer sets a priority weight from `1-5` for each
   category:
   - Working Style
   - Fit
   - Communication
   - Transparency
2. **Core questions**: Consumer answers `16` questions across the four
   categories.
3. **Free fit summary**: Consumer receives an immediate ranked agent list. Agent
   identity remains sealed at this stage.
4. **Paywall**: Pricing options:
   - Free fit summary: $0
   - Pax AI Deep Dive upgrade: $20
     - Purpose: verify working-style fit sharpen match precision
     - Consumer answers:
       - `6` core behavioral questions
       - `3` optional behavioral questions
5. **Consumer education**:
   - Purpose:
     - Explain commission clearly
     - Help rehearse negotiation before agent contact
   - Consumer gets access to:
     - NtC education blogs
     - Pax Negotiator
6. **Consumer selection**
   - Consumer reviews ranked list and selects agent(s). At selection, agent
     identity is revealed:
     - name
     - agency
     - contact information

### Agent Flow

- Focus: create profile and become eligible for matched introductions
- Cost: no pilot subscription fee; fee charged on accepted selection
- Outcome: receive consumer introductions and accept or reject them

1. **Category weighting** Agent sets a priority weight from `1-5` for each
   category:
   - Working Style
   - Fit
   - Communication
   - Transparency

2. **Core questions** Agent answers `12` questions across the four categories.

3. **Profile creation** Agent loads profile details including:
   - experience
   - services
   - zip codes served
   - sample agreements
   - Peace Pact signature

4. **Paywall** Pilot pricing structure:
   - no subscription fee
   - selection fee charged when consumer pick is accepted

5. **Pax AI Deep Dive** Agent completes behavioral profiling to confirm:
   - working style
   - communication approach
   - transparency posture

6. **Platform active** Agent profile goes live and becomes eligible to appear in
   consumer match results.

7. **Agent acceptance and selection fee** Agent reviews consumer profile and
   accepts or rejects introduction.

   Selection fee at acceptance:
   - `$399` when consumer selects `1` agent
   - `$199` per agent when consumer selects up to `3` agents
