# @aamini-stack Monorepo

## Code Style

- CRITICAL: USE PNPM!!! (NOT npm!!!)
- Prefer 'dirty' code. I'm not in the mood for Uncle Bob style of breaking a
  simple problem into 500 microfunctions.

## Verification

After completing a task, run `pnpm verify` in the relevant app directory to ensure
all checks pass (build, lint, format, typecheck, test, e2e).
